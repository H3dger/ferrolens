# Changelog

All notable changes to `ferrolens` will be documented in this file.

The format is inspired by Keep a Changelog and the project uses Semantic Versioning for public releases.

## [0.1.0] - 2026-03-25

### Added

- Terminal-first TUI for reviewing `csv`, `tsv`, `txt`, `vcf`, and `vcf.gz` files
- Table-first layout with horizontal panning and compact detail pane
- Search, filter, manual sort, focused-column sort, reset, export, and hide-column flows
- Current-column-aware filtering prompts and lightweight categorical value hints
- Catppuccin Mocha theme with restrained full-palette accents

### Notes

- This is an early public release aimed at community preview and workflow feedback
- Large-file handling is still memory-first and should be treated as an active improvement area
