# ferrolens Roadmap

## Near term

- Improve large-file strategy for wide and high-row-count result tables
- Expand CSV/TSV edge-case coverage with more real-world fixtures
- Add release assets: screenshots, GIF demo, packaged binaries
- Refine help, onboarding, and first-run discoverability

## Medium term

- Add stronger community-facing packaging (`cargo install`, GitHub Releases)
- Improve filter ergonomics for categorical and high-cardinality columns
- Add more polished export options and explicit output path control
- Strengthen VCF-specific review affordances and compatibility

## Longer term

- Explore lazy loading / chunked reading for large cohort-scale tables
- Add configuration support for theme, keybindings, and view preferences
- Consider optional plugin-style format profiles for common bioinformatics outputs

## Explicit non-goals for now

- BAM pileup or IGV-style visualization
- Mouse-heavy workflows
- Full spreadsheet UI parity
- Project orchestration or workflow management
